import { Tab, Tabs } from 'rspress/theme';

# 本地开发与生产构建

安装依赖:

```bash
pnpm install
```

启动开发服务器：

<Tabs>
  <Tab label="React SPA">
```bash
# 启动 React SPA 应用，支持子包热更新，无需提前构建
pnpm dev
```
	</Tab>
  <Tab label="Next.js">
```bash
# 由于 Next.js 暂不支持子包热更新，需要提前构建子包
pnpm build:pkgs
# 启动 Next.js 应用
pnpm dev:next
```
	</Tab>
</Tabs>

构建生产版本：

<Tabs>
  <Tab label="React SPA">
```bash
pnpm build
```
	</Tab>
  <Tab label="Next.js">
```bash
pnpm build:next
```
	</Tab>
</Tabs>

预览生产版本：

<Tabs>
  <Tab label="React SPA">
```bash
pnpm preview
```
	</Tab>
  <Tab label="Next.js">
```bash
pnpm preview:next
```
	</Tab>
</Tabs>
